# GitHub Release Template — v1.0.0

## Satellite Orbit Simulator v1.0.0

Satellite Orbit Simulator is a desktop application built with **Java 17 + JavaFX** for interactive orbital mechanics visualization.

This release includes:
- Newtonian gravity-based orbital simulation
- LEO / MEO / GEO presets
- Elliptical orbit support
- Multi-satellite visualization
- Orbital trails with fading effect
- Earth atmospheric glow and starfield background
- Rocket launch simulation
- Hohmann transfer maneuver visualization
- Real-time telemetry panels
- Windows installer with bundled Java runtime
