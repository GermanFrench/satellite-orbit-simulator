## 3. LinkedIn post (tecnico/profesional)

Comparto una nueva version publica de **Satellite Orbit Simulator**, una desktop app construida con **Java 17, JavaFX, Maven y una arquitectura separada por capas (`model`, `physics`, `simulation`, `ui`)**.

Aspectos destacados del proyecto:
- simulacion orbital basada en gravedad Newtoniana,
- integracion numerica para actualizar posicion y velocidad,
- soporte para orbitas LEO / MEO / GEO,
- orbitas elipticas,
- multiples satelites en una misma escena,
- sistema de lanzamiento con despliegue orbital,
- transferencia Hohmann con `delta-v` y tiempo de transferencia,
- telemetria en tiempo real,
- instalador Windows `.exe` generado con `jpackage`.

Lo diseñe como un proyecto de portfolio que combina visualizacion, fisica aplicada y arquitectura limpia en Java desktop.
